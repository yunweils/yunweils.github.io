    # 1 项目配置

    本篇教程在示例步骤中使用了以下版本的软件。操作时，请您以实际软件版本为准。

    • 操作系统：CentOS 7.9 64位
    • MySQL：5.7.18

    # 2 基本流程

    操作步骤如下：

    1. 准备编译环境
    2. 安装MySQL

    # 2.1 准备编译环境（centos优化已处理）

    # 2.1.1 关闭防火墙
    # 2.1.2 关闭SELinux

    # 2.2 安装mysql

    # 2.2.1 安装依赖
    yum install -y gcc gcc-c++  make cmake
yum install -y libaio-devel

    2
    # 2.2.2 下载mysql二进制安装包

    官网下载地址 https://downloads.mysql.com/archives/community/ (opens new window)

    mkdir -p /home/mysql
cd /home/mysql
#下载二进制文件
wget https://cdn.mysql.com/archives/mysql-5.7/mysql-5.7.18-linux-glibc2.5-x86_64.tar.gz
#解压二进制文件
tar -zxvf mysql-5.7.18-linux-glibc2.5-x86_64.tar.gz

    # 2.2.4 创建相关目录并修改权限

mkdir /home/mysql/mysql_data
#日志路径

#二进制路径
mkdir -p /home/mysql/mysql_binlogs
ln -s mysql-5.7.18-linux-glibc2.5-x86_64 /home/mysql/mysql_base
chown -R mysql.mysql /home/mysql

    # 2.2.5 修改环境变量:
    #添加环境变量
vim /etc/profile
export PATH=/home/mysql/mysql_base/bin:$PATH
#使环境配置生效
source /etc/profile

    # 2.2.6 初始化数据，初始化管理员的临时密码
    /home/mysql/mysql_base/bin/mysqld --defaults-file=/home/mysql/my.cnf --initialize  --user=mysql --basedir=/home/mysql/mysql_base --datadir=/home/mysql/mysql_data
    1

    注意：将原来模式启动mysqld先关闭，然后再用systemd管理。

    vim /etc/systemd/system/mysqld.service 
或者 vim /usr/lib/systemd/system/mysqld.service [一般存放路径]
[Unit]
Description=MySQL Server
Documentation=man:mysqld(8)
Documentation=http://dev.mysql.com/doc/refman/en/using-systemd.html
After=network.target
After=syslog.target
[Install]
WantedBy=multi-user.target
[Service]
User=mysql
Group=mysql
ExecStart=/home/mysql/mysql_base/bin/mysqld --defaults-file=/home/mysql/my.cnf
LimitNOFILE = 5000

systemctl daemon-reload

##启动mysql
systemctl  start  mysqld

    # 2.2.7 设置密码
    # 2.2.7.1 方法一：查看临时密码
    grep -r "password" /home/mysql/mysql_logs/error.log
    1
    ##重置初始密码为123456
/home/mysql/mysql_base/bin/mysqld -uroot -p临时密码 -S mysql.sock -e "alter user root@'localhost' identified by '123456'"
    1
    2
    # 2.2.7.2 方法二：通过mysqladmin设置密码
    /home/mysql/mysql_base/bin/mysqladmin -uroot -p\"`awk '/A temporary password/ {print $NF}' /home/mysql/mysql_logs/error.log`\" -S /home/mysql/m
